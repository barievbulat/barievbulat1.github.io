Test Case SH-78-36: Загрузка фото для профиля исполнителя [Version : 1]
Author:Bariev_Bulat - 12/10/2020 15:18:16
Summary:

Проверка загрузки файла изображения исполнителя.

Preconditions:

Должен быть создан аккаунт пользователя (см. тест-кейс "Регистрация")

Аккаунт должен быть настроен, как исполнитель услуг.

Должен быть подготовлен графический растровый файл (например с расширением, *.jpg, *.png и т.п.) Максимальный размер файла 15МБ. Минимальное разрешение 400x400.

#:Step actions:Expected Results:
1

Открыть сайт https://yandex.ru/uslugi/

2

Кликнуть на кнопку "Войти" и авторизоваться (логин: testovichtesttt пароль: V25-ABc-Xsm-RD2).

3

Перейти в личный кабинет исполнителя.

4

Перейти в раздел с информацией о профиле исполнителе.

5

Кликнуть "Фото профиля".

6

На открывшейся странице нажать кнопку "Изменить" и выбрать графический растровый файл (например, файл с расширением *.jpg, *.bmp, *.png).

Происходит загрузка файла на сервер, с отображением индикатора в виде вращающегося кружка. По окончанию загрузки индикатор исчезает. Загруженное фото отображается рядом с блоком с текстовой рекомендацией по выбору файла.
